liteScraper

Privacy Policy

Last updated: June 16, 2026

The short version: liteScraper reads files on your Mac, looks up movie metadata from TMDb (and optionally fanart.tv), and writes the results back to your local library. It collects no personal data and stores nothing on any external server.

What liteScraper Does

liteScraper is a local macOS utility. You point it at a folder of movie files on your Mac and it:

  • Reads video filenames to parse movie titles and years
  • Queries The Movie Database (TMDb) for metadata and artwork using an API key you provide
  • Optionally queries fanart.tv for additional artwork using an API key you provide
  • Renames and organizes files on your Mac, writes NFO metadata files, and downloads artwork — all into the folder you selected

Data Stored Locally

liteScraper stores the following data on your Mac only:

  • A security-scoped bookmark to the movie folder you selected (in UserDefaults), used to re-open the same folder across sessions
  • Your TMDb API key (stored in Keychain)
  • Your fanart.tv API key, if provided (stored in Keychain)
  • App preferences such as NFO options and the trailer-move toggle (in UserDefaults)

Network Requests

liteScraper makes outgoing HTTPS requests to two services, both controlled by you:

  • TMDb — to search for movies and fetch metadata, using the API key you provide
  • fanart.tv — to fetch premium artwork, only if you supply a fanart.tv API key

These requests contain only the movie title, year, and TMDb movie ID needed to retrieve metadata. No personal information is included. liteScraper does not communicate with any other server.

What We Don't Collect

We do not collect, transmit, or store any of the following:

  • Personal information of any kind
  • Your movie library contents or file paths
  • Analytics, usage data, or crash reports
  • Advertising identifiers

Your API Keys

The TMDb and fanart.tv API keys you provide are stored exclusively in your Mac's Keychain and are only used to make requests to those respective services. They are never sent to us or any other party.

Children's Privacy

liteScraper does not knowingly collect any information from anyone, including children under the age of 13.
